# 大腸直腸肛門外科 黃政義 品牌連結頁

這個資料夾是可直接上線到 GitHub Pages 的靜態網站包。

## 檔案

- `index.html`：品牌連結頁主檔
- `assets/dr-huang.jpg`：醫師照片
- `.nojekyll`：讓 GitHub Pages 直接以靜態檔案發布

## 建議 GitHub Pages 設定

1. 建立 GitHub repo，建議名稱：`dr-huang-links`
2. 將本資料夾內容放到 repo 根目錄。
3. 到 GitHub repo 的 `Settings` -> `Pages`。
4. Source 選 `Deploy from a branch`。
5. Branch 選 `main`，資料夾選 `/root`。
6. 等待 GitHub Pages 產生公開網址。

預期網址格式通常為：

```text
https://你的GitHub帳號.github.io/dr-huang-links/
```

取得正式網址後，回到專案內替換：

```text
品牌連結頁入口URL_待上線
```
